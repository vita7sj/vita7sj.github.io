| 快捷键                 | 说明              |
| ---------------------- | ----------------- |
| Ctrl + N               | 新建文档          |
| Ctrl + O               | 打开文件          |
| Ctrl + Shift + N       | 新建图层-有提示框 |
| Ctrl + Shift + Alt + N | 新建图层-无提示   |
| Ctrl + Alt + i |打开图像大小|
| Ctrl + L | 色阶 |
| Ctrl + M | 曲线 |
| Ctrl + U  | 色相饱和度 |
|Ctrl + B | 色彩平衡 |
| Ctrl + Y | 检查是否溢色，颜色在RGB模式和CMYK模式下的色彩，溢色无法印刷 |
| Ctrl + Shift + Alt + K | 设置快捷键 |
| Ctrl + G | 图层编组 |
| 英文状态下字母 X | 切换前景色和后景色 |
| Ctrl + delete | 填充后景色 |
| Alt + delete |填充前景色 |
|
